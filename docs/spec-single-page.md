# 体育照明单页站点 · 架构文档

> **参考站点**：Musco / Cooper-Ephesus  
> **设计基调**：深色主题 + 运动场地场景 + 专业工业风  
> **核心原则**：先做最简架构，静态+动画为主，视频组件暂缓

---

## 1. 设计语言

### 1.1 视觉风格
- **风格**：Industrial Dark（工业深色）
- **基调**：夜间运动场地、专业赛事级照明、高端感

### 1.2 配色方案
| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#0A0A0A` | 近黑色 |
| 次背景 | `#1A1A1A` | 深灰卡片 |
| 主色调 | `#F5A623` | 暖黄色（代表光源/能量） |
| 强调色 | `#FFFFFF` | 白色文字/高亮 |
| 辅助色 | `#888888` | 次要文字 |

### 1.3 字体
- **标题**：Montserrat Bold（Google Fonts）
- **正文**：Inter Regular（Google Fonts）
- **备选**：system-ui, sans-serif

### 1.4 间距系统
- 基础单位：8px
- 常用间距：16px / 24px / 48px / 64px / 96px

## 3. 组件清单

### 3.1 Header
- 固定定位，滚动时背景模糊
- Logo（文字或图片占位）
- 导航链接：Products / Solutions / About / Contact
- CTA按钮：Get a Quote

### 3.2 Hero Section
- 全屏高度 (100vh)
- 渐变背景动画（模拟光效）
- 主标题 + 副标题
- 主CTA按钮

### 3.3 Feature Card
- 图标（SVG或Unicode）
- 标题
- 描述文字
- 悬停时上浮 + 边框高亮

### 3.4 Product Card
- 产品图片占位（后续替换）
- 产品名称
- 简短描述
- 悬停：图片放大 + 按钮显现

### 3.5 Stats Counter
- 大数字
- 单位
- 描述标签
- 进入视口时触发数字滚动动画

### 3.6 Contact Form
- Name / Email / Phone / Message
- Submit按钮
- 基础验证

## 4. 动画方案

### 4.1 页面载入动画
- 元素依次淡入上移（stagger 100ms）

### 4.2 滚动触发动画
- Intersection Observer 监测元素进入视口
- 触发：fadeInUp / scaleIn / counter动画

### 4.3 悬停动效
- 卡片：transform: translateY(-8px)
- 按钮：背景色渐变 + 轻微放大
- 链接：下划线动画

### 4.4 Hero背景动画
- CSS渐变动画模拟光束/光晕效果
- 性能优先，不使用视频

---

## 5. 技术实现

### 5.1 技术栈
- **HTML5** 单文件结构
- **CSS3** 内联样式 + CSS变量
- **Vanilla JS** 最小化依赖
- 无框架依赖，最大化加载速度

### 5.2 性能目标
- 首屏加载 < 1.5s
- 无外部视频资源
- 图片使用占位符或CSS渐变
- 所有动画使用transform/opacity

### 5.3 文件结构
```
index.html          # 单页主文件（内联CSS/JS）
```
